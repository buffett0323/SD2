# Demo Presentation

Open `index.html` in a browser to present the project as a slide deck.

Controls:

- Right arrow, space, or `Next`: next slide
- Left arrow or `Prev`: previous slide
- Home / End: first or last slide

Suggested flow follows the video timeline:

1. Title and takeaway
2. Motivation and problem
3. Prior work
4. Task, data, and metrics
5. Method overview
6. Dgrammar
7. DPGrammar
8. Case study
9. Results
10. Discussion and limitations
11. Conclusion
